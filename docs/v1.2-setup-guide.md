# PromptLens v1.2 Setup Guide
## Quick Start with Ollama Integration

---

## ⚡ 5-Minute Setup

### Step 1: Open PromptLens
```bash
# Just open PromptLense.html in your browser
# No server needed - runs entirely client-side
open PromptLense.html
# or
start PromptLense.html
# or
xdg-open PromptLense.html
```

### Step 2: Install Ollama (Optional but Recommended)

**macOS:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

**Linux:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

**Windows:**
Download from https://ollama.com/download

### Step 3: Download a Model

```bash
# Small and fast (recommended for testing)
ollama pull llama3.2

# Or larger models for better analysis
ollama pull mistral
ollama pull qwen2.5
```

### Step 4: Start Ollama

```bash
ollama serve
```

Leave this running in a terminal window.

### Step 5: Test Integration

1. Open PromptLens (PromptLense.html)
2. Type any prompt
3. Click **"🧠 Analyze Output"** button
4. You should see dimension scores and reasoning

---

## 🎯 Using the Feedback System

### Without Ollama (Basic)

1. Type a prompt
2. Look at the score and analysis
3. Click **👍** if it looks correct
4. Click **👎** if it looks wrong
5. Your feedback improves the model over time

### With Ollama (Advanced)

1. Type a prompt
2. Click **"🧠 Analyze Output"**
3. Local LLM evaluates your prompt
4. See detailed breakdown:
   ```
   Overall: 82/100
   clarity: 9/10
   conciseness: 7/10
   structure: 8/10
   
   Reasoning: The prompt is clear and well-structured...
   ```
5. Rate with 👍/👎 to improve future scores

## 📊 Understanding the Display

### Score Display
```
[Score Ring: 75]
Strong (Experimental - correlation not validated)
⚠️ Learned model (heuristic: 73) — correlation being validated

Was this helpful?
👍 (3)  👎 (1)  🧠 Analyze Output

Score: 75 (68-82)
MEDIUM CONF  n=156
```

**What this means:**
- **75**: Your learned model score
- **(68-82)**: 95% confidence interval
- **MEDIUM CONF**: Based on 30-100 samples
- **n=156**: Total feedback samples collected
- **heuristic: 73**: Old formula score (for comparison)

## 🧪 Troubleshooting

### "Ollama not found" error

**Problem:** PromptLens can't connect to Ollama  
**Solution:**
```bash
# Check if Ollama is running
curl http://localhost:11434/api/tags

# If not, start it
ollama serve

# Check your model is downloaded
ollama list
```

### Feedback not saving

**Problem:** localStorage might be disabled  
**Solution:**
- Enable cookies/localStorage in browser settings
- Check privacy mode (often blocks storage)
- Try different browser

### Weights not updating

**Problem:** Need more feedback samples  
**Solution:**
```javascript
// Check current count
getFeedbackCount()

// Need 50+ for weight updates
// Keep rating prompts!
```

### Score seems wrong

**Expected:** Initially scores match heuristic (40 + positive×80...)  
**Improvement:** After 50+ ratings, scores adapt to your preferences  
**Check:** Open console, see if weights have changed from baseline

## 🎁 Tips for Best Results

1. **Rate consistently** - Be honest about what works
2. **Use Ollama** for ground-truth validation on important prompts
3. **Import/Export** to merge data across devices
4. **Share feedback data** with team for collective learning
5. **Check confidence** - Low confidence = experimental score

## 🔒 Privacy Note

**Everything is local:**
- ✅ Feedback stored in browser localStorage
- ✅ Ollama runs on your machine
- ✅ No data sent to external servers
- ✅ No API keys needed
- ✅ No accounts required

**To clear all data:**
```javascript
// In console
localStorage.removeItem('promptLensFeedback');
localStorage.removeItem('promptLensWeights');
location.reload();
```
